# Compatibility Readiness

This document separates the compatibility-related paths that Agent Estate may eventually pursue.

Agent Estate is not currently certified. It is not an official eGovFrame compatibility-confirmed product, and it does not replace eGovFrame compatibility confirmation.

## Contribution recognition

Contribution recognition can include participation records, acknowledgements, awards, or contribution history.

Usefulness:

- public proof of activity,
- early trust signal,
- useful project history.

Limits:

- not product certification,
- not compatibility confirmation,
- not official runtime inclusion.

Agent Estate can pursue this only after it has a working local MVP and clear public messaging.

## Official repository inclusion

Official repository inclusion means a guide, sample, or related artifact is referenced by or accepted into an official eGovFrame repository.

Possible future fit:

- sample documentation,
- VS Code workflow guidance,
- AI/MCP governance checklist,
- public-sector development environment review note.

Limits:

- a documentation reference is not the same as compatibility confirmation,
- official repository inclusion should not be described as certification,
- Agent Estate must keep its local-first and secret-safe boundaries.

## Compatibility confirmation

Compatibility confirmation is a separate official process. It should be considered only when Agent Estate has matured beyond local MVP workflow documentation.

Possible later positioning:

- development support tooling,
- ALM-like review support,
- governance/reporting support for AI/MCP development environments.

Before considering compatibility confirmation:

1. Confirm the relevant software category.
2. Prepare reproducible build and run evidence.
3. Prepare eGovFrame project sample evidence.
4. Prepare security and data-handling boundaries.
5. Prepare support and maintenance notes.

## Messaging Rules

Allowed language:

- "supports eGovFrame 5.0 development workflows"
- "provides a local AI/MCP governance report"
- "helps prepare review artifacts for public-sector development environments"

Avoid:

- "official eGovFrame certified"
- "compatibility confirmed"
- "automatic compliance"
- "security replacement"
- "approved by eGovFrame"

## Current Position

Agent Estate is a local governance and reporting MVP. Its next readiness step is stronger build/runtime evidence and public-facing documentation, not compatibility confirmation.
